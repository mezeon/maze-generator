Usage: python maze-generator.py [size] [offshoots]
size - size of the maze, output image will have size (size, size + 1)
offshoots - number of offshoots, which the maze will have
The default values are size = 75 and offshoots = 300.

Note: Generation of big mazes may take a lot of time.
